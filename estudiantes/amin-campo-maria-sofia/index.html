¿Qué desayuno hoy?

Maria Sofia Amin

Este programa interactivo funciona como un generador visual de desayunos para la semana. El usuario puede hacer clic sobre cada día de la semana (de lunes a viernes) y el sistema selecciona aleatoriamente un desayuno diferente, evitando repeticiones entre los días. Cada opción se representa con ilustraciones dibujadas en p5.js, creando una interfaz dinámica y divertida. Además, el usuario puede modificar el tamaño de las ilustraciones mediante un slider y reiniciar toda la planificación semanal con un botón, permitiendo explorar nuevas combinaciones de desayunos de manera rápida.
